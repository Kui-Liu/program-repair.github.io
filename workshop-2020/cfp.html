1st International Workshop on Automated Program Repair (APR 2020)

We are pleased to announce the 1st International Workshop on Automated Program Repair (APR 2020) co-located with International Conference on Software Engineering (ICSE 2020), Seoul, South Korea. Automated program repair is an emerging research area that aims to reduce the time and effort spent on fixing software bugs. The recent explosive growth of works on program repair has culminated in successful deployments of program repair in industry. Despite these considerable advances, program repair still faces fundamental technological and societal challenges. This workshop will create an opportunity for the research community to coordinate its effort for addressing these challenges, share recent ideas and results, and formulate an agenda for future research. More information is available on the workshop website:
http://program-repair.org/workshop-2020/
Topics of Interest
  • Quality of automatically generated patches
  • Scalability of program repair algorithms
  • Repair of domain-specific bugs
  • Applications of program repair
  • Deployment of program repair
  • Case studies on program repair
  • Program repair tools
  • Program repair benchmarks
  • Program analysis for program repair
  • Machine learning of code for program repair
  • Testing and verification for program repair
  • Fault localization for program repair
  • Debugging and program repair
  • Program synthesis for program repair
